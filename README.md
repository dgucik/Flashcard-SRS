# Flashcard SRS

## Overview
A web-based platform for learning with flashcards using spaced repetition (SRS). Users can create, study, and browse Sets, including language Sets with translation hints.

## Project Structure
- `docs/` – project documentation including business, system and technical layers.
- `backend/` – backend API and business logic
- `frontend/` – web interface for users

## Features
- User account management (register, login, profile)
- Sets: create, edit, delete, copy
- Flashcards: add, delete within Sets
- SRS-based study sessions with progress tracking
- Browse public Sets
- Language Flashcards with translation hints
