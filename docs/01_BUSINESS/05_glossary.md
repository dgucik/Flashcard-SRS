# Glossary

## Terms
| Term | Definition |
|------|------------|
| User | A person who uses the system, owns an account, and performs actions within the application. |
| Profile | The user's page containing personal data and settings (e.g., password change). |
| Space | The user's private area containing their sets. |
| Set | A collection of flashcards created or copied by the user, containing a title, description, tags, and flashcards. |
| Flashcard | A single learning card with two sides (question/answer or word/translation). |
| Progress | A percentage indicator showing how much of the set has been learned based on flashcard levels. |
| Statistics | Detailed information about a set: number of flashcards per level, study time, activity. |
| Dashboard | A view of a set showing statistics and allowing the user to start a study session. |
| Level | The SRS learning stage of a flashcard (1–5 or “memorized”). |
| Memorized | A status of a flashcard indicating it has been fully learned and has the longest review interval. |
| Session | A study session in which the user reviews flashcards according to the SRS algorithm. |
| Standard | A type of set without language-specific features. |
| Language | A type of set with language features, such as auto-translations. |
| Public Set | A publicly available set that any user can browse or copy. |
| Translation Hint | An automatically generated translation suggestion for language flashcards. |
| Review | A scheduled repetition of flashcards based on their SRS levels. |
