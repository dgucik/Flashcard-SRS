# Glossary

## Terms
| Term | Definition |
|------|------------|
| Flashcard | A single learning item containing a question and an answer, used for studying. |
| Set | A group of flashcards created by a user for study purposes. |
| User / End User | A person who interacts with the platform to create, study, or browse flashcard sets. |
| Public Set | A flashcard set available to all users for browsing and copying. |
| Copied Set | A duplicate of a public set that becomes fully independent and owned by the copying user. |
| Progress | A metric showing how much of a set a user has learned, from 0% (all level 1) to 100% (all memorized). |
| Session | A learning activity where the user goes through flashcards of a set. |
| Dashboard (User) | The initial dashboard showing the user’s own sets and copied sets with basic info and overall progress. |
| Dashboard (Set) | A detailed view for a specific set showing flashcards per level, weekly study time, and option to start a study session. |
| Translation Hint | Suggested translation or example sentence for language-type flashcards. |
| Level | A numeric indicator of a flashcard’s current mastery stage in the learning system (1–5, memorized). |
