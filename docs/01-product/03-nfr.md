# 3. Technical & Non-Functional Requirements

## 3.1. Performance & UX
* **PERF-01:** System/API response time must be under **1 second** (1000ms).
* **UX-01:** Interface must be fully responsive (Desktop & Mobile).

## 3.2. Security & Reliability
* **SEC-01:** Passwords stored using strong hashing.
* **LOG-01:** Centralized error logging for rapid diagnosis.
